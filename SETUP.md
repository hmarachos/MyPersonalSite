## 📁 Структура проекта

```
project/
├── index.html          # Основной HTML файл
```

## 🎯 Как запустить

### Вариант 1: Просто откройте файл
1. Найдите файл `index.html` в папке проекта
2. Дважды кликните на него или откройте через браузер
3. Готово! Сайт загружен и работает

### Вариант 2: Локальный сервер (рекомендуется)
Если вы хотите избежать проблем с CORS при добавлении изображений:

**На macOS/Linux:**
```bash
# Python 3
python3 -m http.server 8000

# Или Python 2
python -m SimpleHTTPServer 8000
```

**На Windows:**
```bash
# Python 3
python -m http.server 8000
```

Затем откройте в браузере: `http://localhost:8000`

## 🎨 Как изменить цвета

**Файл:** `style.css`  
**Строки:** 5-20

```css
:root {
    --primary: #2563eb;           /* Основной синий */
    --primary-dark: #1e40af;      /* Тёмный синий */
    --text-dark: #1f2937;         /* Тёмный текст */
    --text-light: #6b7280;        /* Светлый текст */
    --bg-light: #f9fafb;          /* Светлый фон */
    --bg-white: #ffffff;          /* Белый фон */
}
```

Примеры цветов:
- Синий: `#2563eb`
- Зелёный: `#10b981`
- Красный: `#ef4444`
- Оранжевый: `#f97316`
- Фиолетовый: `#8b5cf6`

Сайт автоматически адаптируется под:
- 📱 Мобильные телефоны (320px+)
- 📱 Планшеты (768px+)

Точки разрыва в `style.css`:
- `@media (max-width: 768px)` — планшеты
- `@media (max-width: 480px)` — мобильные

## 🔧 Добавление изображений

### Создайте папку `assets`
```bash
mkdir assets
```

### Добавьте изображение в HTML
**Пример для Hero:**
```html
<section class="hero">
    <div class="container">
        <div class="hero-content">
            <!-- Текст -->
        </div>
        <img src="assets/hero-image.png" alt="AI для инженерии" class="hero-image">
    </div>
</section>
```

**Добавьте стили в `style.css`:**
```css
.hero-image {
    max-width: 400px;
    width: 100%;
    border-radius: 12px;
}

@media (max-width: 768px) {
    .hero-image {
        max-width: 100%;
        margin-top: var(--spacing-xl);
    }
}
```

## 📧 Интеграция формы с сервером

Сейчас форма работает локально. Чтобы отправлять данные на сервер:

**Вариант 1: Использовать Formspree (бесплатно)**

1. Перейдите на https://formspree.io
2. Создайте новую форму
3. Замените в `index.html` атрибут `action`:

```html
<form class="contact-form" action="https://formspree.io/f/YOUR_ID" method="POST">
    <div class="form-group">
        <input type="text" name="name" placeholder="Ваше имя" required>
    </div>
    <div class="form-group">
        <input type="email" name="email" placeholder="Email" required>
    </div>
    <button type="submit" class="btn btn-primary btn-large">Отправить</button>
</form>
```

**Вариант 2: Использовать собственный сервер**

Раскомментируйте код в `script.js` (строки 30-36) и создайте API endpoint.
